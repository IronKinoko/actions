# actions

自动化脚本

用`typescript`与`node`配合`Github Actions`实现自动化

## 环境变量

| 参数             | 类型      | 说明                         |
| ---------------- | --------- | ---------------------------- |
| `GITHUB_ACTIONS` | `boolean` | 是否在`Github Actions`中运行 |
| `FL`             | `string`  | fl登录账号密码               |
| `BARK_URL`       | `string`  | bark推送url                  |
| `YS_COOKIE`      | `string`  | 原神cookie 用于网页活动签到  |
